Что выведет программа? Объяснить вывод программы.

```go
package main
 
import (
    "fmt"
)
 
func main() {
    a := [5]int{76, 77, 78, 79, 80}
    var b []int = a[1:4]
    fmt.Println(b)
}
```

Ответ:
```
[77 78 79]
```

[//]: # (Берется срез исходного массива 
с 1 по 4 ,не включительно, элемент, 
в результате получается слайс из 3 элементов)
